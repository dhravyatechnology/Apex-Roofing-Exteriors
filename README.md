# Apex Roofing & Exteriors

Premium Next.js landing page for Apex Roofing & Exteriors, a bold roofing and exterior services company serving Tampa, Florida.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- `next/image`
- `next/font`

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Checks

```bash
npm run lint
npm run build
```

## SEO

The project includes:

- Metadata API
- Canonical URL
- Open Graph and Twitter card metadata
- `robots.ts`
- `sitemap.ts`
- Organization schema
- Roofing contractor local business schema
- FAQ schema
- Semantic landing page structure

## Deployment

### Vercel

1. Import this project into Vercel.
2. Keep the default framework preset as Next.js.
3. Build command: `npm run build`
4. Output is handled automatically by Vercel.

### Netlify

1. Import this project into Netlify.
2. Build command: `npm run build`
3. Publish directory: `.next`
4. Use the official Netlify Next.js runtime/plugin when prompted.
